Running Locally

  • npm serve - open http://localhost:8080/lessons/
  • npm run clean - runs `node tools/clean-generated.mjs`
  • npm run build-lessons- runs `node tools/generate-lessons-site.mjs`
  • npm serve- runs `npx http-server public -p 8080`
